# Aplia Landing Page

Este projeto contém o código fonte da Landing Page da Aplia.

## Estrutura
- `index.html`: Arquivo principal do site.
- `.github/workflows/deploy.yml`: Configuração de deploy automático via FTP.

## Como editar
1. Faça as alterações nos arquivos locais.
2. Envie para o GitHub (`git push`).
3. O deploy será feito automaticamente para a Locaweb.
